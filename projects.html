Programming Projects:

#1 The Yurt Index

This is my most recent project. It is an Irish slang dictionary called the Yurt Index. It was made as a Milestone Project for the Code Institute. It was made with the Flask micro-framework for Python. I am very happy with the final version, but there are some upgrades planned in the future.

#2 Anagrammer

This was my second Milestone Project for the Code Institute. It's a word formation game made primarily with JavaScript. While I am very proud of the robust functionality of the game, some UX changes are planned in the near future!

#3 Michael Walsh

The website you're reading this from! This was the first website that I made on my own, and while it was my first Milestone Project for the Code Institute, it was one that I built completely from scratch. The site is meant to display my ability to build a website independently and showcase my abilities with CSS and HTML.

Writing Projects:

#4 Before the Fall, Sample Chapter 1

#5 Before the Fall, Sample Chapter 2
